# ✈️ Travel Assistant — IBM SkillsBuild AICTE Virtual Internship

> A conversational AI travel planning agent built with **IBM watsonx Orchestrate** and powered by the **IBM Granite** foundation model.

---

## 📌 Project Overview

The **Travel Assistant** is an AI-powered conversational agent that helps users plan personalised trips through a natural chat interface. Instead of juggling multiple travel websites, users simply describe their preferences and receive tailored recommendations in one conversation.

| Detail | Value |
|---|---|
| **Platform** | IBM watsonx Orchestrate |
| **Foundation Model** | IBM Granite (via watsonx Orchestrate) |
| **Project Type** | IBM SkillsBuild AICTE Virtual Internship |
| **External APIs** | None (AI-driven recommendations) |
| **Deployment** | IBM SkillsBuild Environment |

---

## 🎯 Problem Statement

Travel planning is fragmented and time-consuming. Users must visit multiple platforms — flight aggregators, hotel booking sites, itinerary planners, and travel blogs — before they can make a decision. This project consolidates that experience into a single conversational AI agent that understands user preferences and generates personalised guidance instantly.

---

## ✨ Features

### 1. 🗺️ Destination Suggestions
The assistant collects the user's travel interests (adventure, culture, relaxation, food, etc.) and budget to recommend suitable destinations that match their profile.

### 2. 🏨 Hotel Recommendations
Based on the user's destination, budget, and travel style, the assistant suggests hotel types and accommodation options suited to their needs.

### 3. 📅 Day-by-Day Itinerary Generation
The assistant creates a structured day-by-day travel plan covering activities, sightseeing, meals, and rest time — tailored to the trip duration provided by the user.

### 4. 🎒 Personalised Travel Checklist
Generates a packing and preparation checklist customised to the destination, season, trip duration, and travel type (solo, family, couple, etc.).

### 5. 💡 Travel Tips & General Q&A
Answers general travel-related questions covering visa requirements, local customs, safety tips, currency advice, and transport options.

---

## 💬 Conversational Flow

The agent follows a structured conversation flow before generating any recommendation:

```
1. Greet the user and introduce the assistant
2. Ask for destination (or suggest if the user is unsure)
3. Collect travel dates / trip duration
4. Ask about budget range
5. Ask for number of travellers and travel type (solo / couple / family / group)
6. Understand travel interests (adventure / culture / food / relaxation / shopping)
7. Generate personalised recommendations based on collected inputs
8. Offer itinerary, hotel suggestions, and packing checklist
9. Answer any follow-up questions
```

## ⚠️ Current Limitations

- Does **not** perform actual bookings (flights, hotels, or activities).
- Recommendations are AI-generated and **not sourced from live data**.
- No real-time pricing, availability, or weather information.
- Limited to English language interactions.
